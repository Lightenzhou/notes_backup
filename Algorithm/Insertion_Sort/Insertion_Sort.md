## 代码实现

对于排序算法，优先完成单趟排序。对插入排序而言，我们仅需将插入的该数据记录，依次从末尾进行比较，若比对应数据小则把对应数据往后移，若比对应数据大则插入数据。（以整型为例）

```cpp
int end; //记录尾位置
int tmp; //保存需要插入的数据

while (end >= 0) //比到首元素为止
{
	if (tmp > arr[end]) //大则插入数据
    {
        //如何插入？
    }
    else //小则后移对应数据
    {
        arr[end + 1] = arr[end];
    }

    end--;
}
```

这里需要注意的是，插入有一种特殊情况，若该数是最小的，则 pos 会走到 -1 的位置跳出循环，此时插入的数据应在 0 位置，而这和大则插入数据的执行逻辑一致，也就是在 pos + 1 的位置插入，故这两种情况同时处理。如果不同时处理则需要在跳出循环后判断是正常结束还是因为 pos 到达 -1 结束，逻辑执行不完美。

```cpp
int end;
int tmp;

while (end >= 0)
{
	if (tmp > arr[end])
    {
        break;
    }
    else
    {
        arr[end + 1] = arr[end];
    }

    end--;
}

arr[end + 1] = tmp; //*
```

而整个循环第一次是不需要进行插入排序的，所以循环从 1 开始，且需要排完数组的所有元素，则到 n 结束。（从 0 开始也是可行的，不做演示）

```cpp
void insertion_sort(int* arr, int n)
{
	for (int i = 1; i < n; ++i) //从 1 开始
	{
		int end = i - 1; //第一个开始比较的位置是需要插入的数据之前的位置
		int tmp = arr[i]; //需要插入的数据

		while (end >= 0)
		{
			if (tmp > arr[end])
			{
				break;
			}
			else
			{
				arr[end + 1] = arr[end];
			}

			end--;
		}

		arr[end + 1] = tmp;
	}
}
```

## 效率分析

插入排序时间复杂度：`O(N^2)`

第一次插入最坏情况执行`1`次，第二次插入最坏情况执行`2`次......第`n`次插入排序最坏情况执行`n`次。
则其为一个等差数列：`Sn = 1 + 2 + 3 + ... + n - 1 + n = (1 + n) * n / 2 = 1/2 * (n^2 + n) `
可忽略常数`1/2`和阶数较低的`n`，故得时间复杂度为：`O(N^2)`

## 补充说明

* 本文插入排序中比较方是从后往前比，从前往后比理论上不可行，例如：5 7 这两个数，现在插入 1 这个数，从前往后比然后后移数据会导致数据覆盖，必须从后往前比，先移走 7 再移走 5。

* 插入排序是稳定的排序算法，因为在插入过程中存在相对数据是不必交换的，可以保证相同数据的相对位置相同。
