大目录命名规则 - 首字母大写 + 不分割

小目录命名规则（过长时必要单词可连写分割）：
Algorithm - 首字母大写 + 下划线分割单词
DataStructure - 首字母大写 + 下划线分割单词
Linux - 首字母大写 + 下划线分割单词
Practice - 首字母大写 + 不分割

all in all：有具体规范的大写 + 分割，无规范的大写 + 连写

Algorithm - md 书写规则
1.算法介绍 
2.代码实现 （命名除固定数据结构外不强加要求，但需首字母小写 + 下划线分割单词）
3.效率分析（数学推导整个过程严格遵守行内代码 `` 规则） 
4.补充说明（可省）

DataStructure - md 书写规则
1.结构类
2.成员函数（首字母小写 + 下划线分割单词，数据结构名称（可简写）+ 函数功能（尽量遵循 STL））
3.效率分析
4.补充说明（可省） - 补充说明中大点位于前，注解位于后（[^]: 的内容为注解）

Practice - md 书写规则
1.模型分析
2.问题引出
3.问题剖析
4.效率分析（对不同解决方法的效率分析，仅模型分析时可选）
5.补充说明（可省）

Linux - md 书写规则
无具体规则，依据实际情况讲解即可。
